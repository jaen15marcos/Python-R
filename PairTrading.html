Statistical Arbitrage and Pair Trading

Statistical Arbitrage involves using statistical methods to identify trading opportunities based on mean reversion. It leverages short-term price movements and statistical properties of time series data.

Pair Trading is a type of statistical arbitrage where two correlated securities are traded simultaneously. The idea is to exploit the relative price movements of the pair:

1. Mean Reversion

If two securities are cointegrated, deviations from their historical relationship are expected to revert to the mean. Traders can short the overperforming security and long the underperforming one.

2. Spread

The difference between the prices of the two securities. When the spread deviates significantly from its mean, a trade is initiated based on the expectation that the spread will revert.

Key Concepts:

1. Stationarity: A stationary time series has a constant mean, variance, and autocorrelation structure over time.

2. Non-Stationarity: Most financial time series are non-stationary, meaning their statistical properties change over time. However, differences between such series can be stationary.

3. Cointegration: Two non-stationary time series $LaTex{\(X_t\)} and $LaTex{\(Y_t\)} are cointegrated if there exists a linear combination $LaTex{\(\alpha X_t + \beta Y_t\)} that is stationary.

4. Johansen Test: A statistical test for cointegration that uses a vector autoregressive model to test for the presence of cointegration relationships among multiple time series.

5. Augmented Dickey-Fuller (ADF) Test: A test for stationarity that checks whether a time series has a unit root. For cointegration, the residuals from the cointegrated series should be stationary.

Summary of the Process

1. Data Preparation

Load the necessary packages. Read the data and filter it to include only relevant rows and columns. Select data for the year 2019 and ensure there are at least 250 trading days for each symbol. Pivot the data to have symbols as columns and dates as rows.

2. Correlation Analysis

Calculate the correlation matrix for the numeric data (closing prices). Identify pairs of symbols with a high correlation (above 0.7) and remove redundant pairs.

3. Cointegration Testing

For each pair of highly correlated symbols, test for cointegration using the Johansen test. If the test indicates cointegration, perform an Augmented Dickey-Fuller (ADF) test on the residuals to confirm stationarity.

4. Filtering Pairs

Merge the identified cointegrated pairs with additional benchmark and currency information. Filter the pairs based on specific criteria such as different benchmarks but the same currency, test benchmarks names similarity using Levenshtein distance.

5. Plotting and Analysis

Plot the time series of the identified pairs. Implement a Kalman filter to dynamically adjust the hedge ratio and intercept. Generate trading signals based on deviations from the mean (standard deviation bands).

6. Backtesting

Simulate positions based on the trading signals. Calculate the P&L (Profit and Loss) from these positions. Plot the cumulative P&L to evaluate the performance of the strategy.

  1. Test for cointegration
# Initialize an empty dataframe to store cointegrated pairs
cointegrated_pairs <- data.frame(pair1 = character(), pair2 = character(), stringsAsFactors = FALSE)

# Test for cointegration
for (i in 1:nrow(df_pairs)) {
  pair <- df_pairs[i, ]
  name1 <- pair[1]
  name2 <- pair[2]

  values1 <- df[[as.character(name1)]]
  values2 <- df[[as.character(name2)]]

  # Perform Johansen cointegration test
  test <- summary(urca::ca.jo(data.frame(values1, values2), type="trace", K=2, ecdet="none", spec="longrun"))

  if (test@teststat[1] > test@cval[5] && test@teststat[2] > test@cval[6]) {
    index <- match(max(test@lambda), test@lambda)
    if (index == 1) {
      s <- values1 * test@V[1] + values2 * test@V[2]
    } else {
      s <- values1 * test@V[3] + values2 * test@V[4]
    }

    cointegration_test_result <- tseries::adf.test(s)
    if (cointegration_test_result$p.value < 0.05) {
      cointegrated_pairs <- rbind(cointegrated_pairs, data.frame(pair1 = as.character(name1), pair2 = as.character(name2), stringsAsFactors = FALSE))
    }
  }
}

  1. Generate Signals
x <- df$XIC
y <- df$VCN
x$int <- rep(1, nrow(x))
delta <- 0.0001
Vw <- delta/(1-delta)*diag(2)
Ve <- 0.001
R <- matrix(rep(0, 4), nrow=2)
P <- matrix(rep(0, 4), nrow=2)
beta <- matrix(rep(0, nrow(y)*2), ncol=2)
y_est <- rep(0, nrow(y))
e <- rep(0, nrow(y))
Q <- rep(0, nrow(y))
for(i in 1:nrow(y)) {
  if(i > 1) {
    beta[i, ] <- beta[i-1, ] # state transition
    R <- P + Vw # state cov prediction
  }
  y_est[i] <- x[i, ] %*% beta[i, ] # measurement prediction
  Q[i] <- x[i, ] %*% R %*% t(x[i, ]) + Ve # measurement variance prediction
  
  # error between observation of y and prediction
  e[i] <- y[i] - y_est[i]
  K <- R %*% t(x[i, ]) / Q[i] # Kalman gain
  
  # state update
  beta[i, ] <- beta[i, ] + K * e[i]
  P = R - K %*% x[i, ] %*% R
}
beta <- as.xts(beta, order.by = zoo::index(df)) #ignore date
plot(beta[2:nrow(beta), 1], type='l', main = 'Kalman updated hedge ratio')

plot(beta[2:nrow(beta), 2], type='l', main = 'Kalman updated intercept')

# Plot trading signals at .75 standard deviation of the prediction error
e <- xts(e, order.by = zoo::index(df))
sqrtQ <- xts(sqrt(Q), order.by = zoo::index(df))
signals <- merge(e, .75 * sqrtQ, .75 * -sqrtQ)
colnames(signals) <- c("e", "sqrtQ", "negsqrtQ")
plot(signals[10:length(zoo::index(signals))], ylab = 'e',

6. Backtest

# Vectorized backtest
sig <- ifelse((signals[1:length(index(signals))]$e > signals[1:length(index(signals))]$sqrtQ) & (lag.xts(signals$e, 1) < lag.xts(signals$sqrtQ, 1)), -1, ifelse((signals[1:length(index(signals))]$e < signals[1:length(index(signals))]$negsqrtQ) & (lag.xts(signals$e, 1) > lag.xts(signals$negsqrtQ, 1)), 1, 0))
colnames(sig) <- "sig"
sig[sig == 0] <- NA
sig <- na.locf(sig)
sig <- diff(sig) / 2
plot(sig)

# Simulate positions and PnL
sim <- merge(lag.xts(sig, 1), beta[, 1], x[, 1], y)
colnames(sim) <- c("sig", "hedge", colnames(x[, 1]), colnames(y[, 1]))
sim$posX <- sim$sig * -1000 * sim$hedge
sim$posY <- sim$sig * 1000
sim$posX[sim$posX == 0] <- NA
sim$posX <- na.locf(sim$posX)
sim$posY[sim$posY == 0] <- NA
sim$posY <- na.locf(sim$posY)
pnlX <- sim$posX * diff(sim[, colnames(x[, 1])])
pnlY <- sim$posY * diff(sim[, colnames(y[, 1])])
pnl <- pnlX + pnlY
